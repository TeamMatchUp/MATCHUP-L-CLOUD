Emotional Thesis
Feels like a professional fight operations headquarters — composed, analytical, and controlled — yet open enough to encourage discovery and opportunity.
Inspired by principles from design-tips.md: start with feeling first .
MatchUp leads with emotional intent before structure.
Discovery feels accessible.
Matchmaking feels official.

4. Spacing & Layout System
8pt grid enforced globally.

Core Spacing Scale
8px (micro spacing)
16px (tight grouping)
24px (component padding)
32px (section separation)
64px+ (major sections)

Layout Principles
Public pages:
Wider margins
More whitespace
Centered content blocks
Organiser dashboard:
Dense grid alignment
Multi-column layout
Compact data rhythm
Coach dashboard:
Slightly more breathing room
Card-based roster layout
Sidebar navigation for authenticated users.
Wide comparison cards for matchup breakdowns.

5. Motion & Interaction (Kindness Applied)
Kindness does not mean softness.
It means controlled feedback.
Inspired by “Prompt behavior, not just state” .

Motion Parameters
Duration: 200–250ms
Easing: ease-in-out
No bounce effects
No playful motion

Interaction Behaviors
Hover:
Subtle elevation (2–4px shadow increase)
Slight border accent
Panels:
Soft slide-in from right
Status Change:
Smooth color transition
No flashing
AI Suggestions:
Subtle gold outline glow
Low-opacity pulse (single cycle only)

6. Voice & Tone
Tone: Strategic. Neutral. Clear.
No hype language.
No exclamation marks.
No emojis.

Microcopy Examples
On proposal creation:
“Match proposal created.”
Pending state:
“Awaiting coach confirmation.”
Pending fighter:
“Fighter response pending.”
Confirmed:
“Match confirmed.”
Error:
“Action unavailable. Status locked.”
Onboarding:
“Create structured, verified matchups across regions.”

7. System Consistency Anchors
Visual anchors:
Grid precision similar to Linear
Component clarity grounded in shadcn/ui
Authority minimalism inspired by financial dashboards
Recurring metaphors:
Control room (Organiser)
Team management console (Coach)
Match card as official contract preview

8. Accessibility
WCAG AA contrast minimum
16px minimum body text
Clear focus ring (Steel Blue outline)
ARIA labels for match status updates
Full keyboard navigability
No status indicated by color alone (include icon + label)

9. Emotional Audit Checklist
Before launch, confirm:
Does this feel composed, not chaotic?
Is gold used sparingly?
Does confirmation feel official?
Does public browsing feel open?
Would an organiser trust this with high-value events?

10. Technical QA Checklist
Typography aligns to 8pt grid
All colors pass AA+
Interactive states visually distinct
Motion stays within 150–300ms
Status states clearly labeled

Emotional Thesis (One Sentence)
A controlled, intelligent fight operations system that feels authoritative without being aggressive.

Design Integrity Review
MatchUp successfully balances openness and control through emotional contrast:
Public discovery feels breathable.
Match confirmation feels structured and official.
Gold is used strategically to signal authority.
Motion reinforces calm confidence.
Improvement suggestion:
Introduce a subtle background texture variation between discovery and dashboard areas to further reinforce psychological context shift without increasing visual noise.
